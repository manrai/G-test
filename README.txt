Installing the BB-EIGHT API:

1. Requirements

Python packages required:
flask
numpy
pandas
sqlite3
Json
subprocess

R packages required:
tidyverse
RSQLite
ggthemes
optparse
stringi

System:
bcftools

2. Running the BB-EIGHT API from the terminal:

$ python3 bb-eight.py

For details on API endpoints and usage, please see the NHLBI BioData Catalyst BB-EIGHT Tutorial in the /tutorial directory.
